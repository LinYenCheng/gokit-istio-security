We make our application accessible from outside of the Kubernetes cluster by using istio gateway, so we need to set gateway url first.
Get the gateway information:
  export INGRESS_HOST=$(kubectl -n istio-system get service istio-ingressgateway -o jsonpath='{.status.loadBalancer.ingress[0].hostname}')

1. Usage for http (Now we use 80 port for the http)
  curl -X "POST" "http://$INGRESS_HOST/api/v1/add/sum" -H 'Content-Type: application/json; charset=utf-8' -d '{ "a": 3, "b": 34}'
  curl -X "POST" "http://$INGRESS_HOST/api/v1/foo/foo" -H 'Content-Type: application/json; charset=utf-8' -d '{"s": "3ddd"}'

2. Usage for grpc (Now we use 443 port for the grpc)
  grpcurl -H 'version: v1' -plaintext -proto ./pb/addsvc/addsvc.proto -d '{"a": 3, "b":5}' $INGRESS_HOST pb.Addsvc.Sum
  grpcurl -plaintext -proto ./pb/foosvc/foosvc.proto -d '{"s": "foo"}' $INGRESS_HOST pb.Foosvc.Foo
